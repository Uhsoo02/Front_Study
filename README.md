# Front_Study  
📚 방학 프론트엔드 학습 기록 (HTML · CSS · JavaScript · React)

---

## 📌 소개
컴퓨터공학 4학년 진입 전 방학 기간 동안  
종합 팀 프로젝트를 대비하여 **프론트엔드 역량 강화를 목표로 진행한 학습 기록**이다.

HTML / CSS / JavaScript 기초부터 시작하여  
React 컴포넌트 구조와 API 연동까지 단계적으로 학습한다.

모든 학습은 **Day 단위 실습 + README 정리** 방식으로 기록한다.

---

## 🎯 학습 목표
- 웹 프론트엔드 기본 구조에 대한 명확한 이해
- JavaScript를 활용한 DOM 조작 및 이벤트 처리 능력 향상
- React 컴포넌트 기반 개발 흐름 이해
- 팀 프로젝트에서 프론트엔드 역할을 수행할 수 있는 실전 감각 습득

---

- GitHub 커밋 로그로 학습 흐름 관리
